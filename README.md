# 📝 Todo List Application

A responsive and interactive **Todo List Application** built using **HTML5, CSS3, and Vanilla JavaScript**. This project helps users efficiently manage their daily tasks by allowing them to add, edit, delete, and store todos using the browser's Local Storage.

## 🚀 Features

* ✅ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* 💾 Persistent storage using Local Storage
* 📱 Responsive user interface
* 🎨 Clean and modern design
* ⚡ Real-time task management

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

1. Clone this repository:

```bash

```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.

No additional installation is required.
